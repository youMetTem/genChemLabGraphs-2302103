# 2302103 General Chemistry Laboratory – Graph Plotting Scripts

This repository contains Python scripts used to generate graphs and perform simple numerical analysis for experiments in **2302103 General Chemistry Laboratory**.

Many experiments in this course require transforming measured data into plots and extracting parameters using **linear regression or model equations**. The scripts here automate those steps and help ensure consistent calculations and visualization.

## Requirements
- Python 3.x  
- `numpy`
- `matplotlib`

```bash
pip install numpy matplotlib
